layout.html 파일은 특정 위치의 내용을 상활별로 달리하지만 양식은 하나로 유지해야할때 사용합니다

여기서부터 확장 내용 시작입니다.

{% block content %} {% endblock %}

여기까지가 확장 내용 끝입니다.

다시 layout.html이 시작합니다

include 는 다른 파일의 내용을 이곳으로 불러와서 표시하는 형식이지만, extends 는 현재파일이 다른 애용에 제공되어 양식을 구성하는 형식입니다. 그래서 app.js 에서 render 할때도 layout.html 파일을 지목하지 않고, block content 의 내용을 이루는 파일을 지목합니다.

layout.html은 양식 파일이며 다른 중요 내용을 제공할 용도로 사용하므니다.
